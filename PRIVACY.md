# Privacy Policy — shirokuma-dev-os

**最終更新: 2026-07-08**

## 日本語

`shirokuma-dev-os` は、開発規律を Claude Code 等の AI エージェントに配給する
スキル・テンプレート・ローカルスクリプトの集合です。

- **個人データを一切収集しません。**
- **テレメトリ・解析・トラッキングを一切含みません。**
- **外部サーバーへの通信を行いません**（同梱スクリプト `init.mjs` /
  `audit-self-integrity.mjs` はローカルのファイル操作のみ）。

したがって、本プラグイン自体が収集・保存・送信するユーザーデータはありません。

> 注: 本プラグインを利用する AI エージェント本体（Claude Code 等）や、
> あなたが別途設定する外部サービスのデータ取り扱いは、それぞれの提供者の
> プライバシーポリシーに従います。本プラグインはそれらに関与しません。

## English

`shirokuma-dev-os` is a collection of skills, templates, and local-only
scripts that deliver engineering discipline to AI agents such as Claude Code.

- It **collects no personal data.**
- It contains **no telemetry, analytics, or tracking.**
- It makes **no network requests** (the bundled `init.mjs` and
  `audit-self-integrity.mjs` scripts perform local file operations only).

The plugin itself collects, stores, and transmits no user data.

> Note: The AI agent host (e.g. Claude Code) and any external services you
> configure separately are governed by their own privacy policies. This
> plugin does not participate in them.
